## EXPLAIN

## [3] AI의 한계 알기
═══════════════════════════════

AI는 강력하지만 완벽하지 않습니다.
한계를 알아야 제대로 쓸 수 있습니다.

### AI가 못하는 것

  🎭 환각 (Hallucination)
  ─────────────────────────────
  확인 안 된 정보를 자신있게 말합니다.
  예: "EXEM은 2023년 매출 1000억을 기록했다" (잘못된 정보)
  AI는 "모른다"고 말하는 것보다 "지어내는" 쪽을 선택하기 쉽습니다.

  🌐 인터넷 접근 불가
  ─────────────────────────────
  실시간 정보를 조회할 수 없습니다. (MCP 연결 전)
  "오늘 주가", "최신 뉴스" 같은 질문에는 답할 수 없습니다.
  → Day 3에서 MCP로 외부 도구를 연결하면 해결됩니다!

  📏 컨텍스트 윈도우 한계
  ─────────────────────────────
  대화가 길어지면 앞부분을 잊습니다.
  → CLAUDE.md가 이 문제를 완화합니다 (영구 기억)

  📅 학습 기준일
  ─────────────────────────────
  특정 날짜 이후의 정보를 모릅니다.
  2026년 3월의 최신 뉴스를 물어봐도 모를 수 있습니다.


### 대응법

  ✅ 검증 습관: AI 결과를 무조건 믿지 않는다
  ✅ 출처 요청: "출처를 밝혀줘"
  ✅ 불확실성 표시: "확실하지 않으면 모른다고 해줘"
  ✅ 단계별 확인: 큰 작업은 쪼개서 각 단계를 확인

"AI를 의심하는 것이 아니라, 확인하는 습관을 만드는 것입니다."

## EXECUTE

### 의도적 실패 체험

AI의 한계를 직접 체험해봅시다.

**체험 1: 인터넷 접근 불가**
아래를 입력해보세요:
```
오늘 EXEM 주가가 얼마야?
```
→ Claude는 실시간 주가를 알 수 없습니다.
→ Day 3에서 MCP를 연결하면 해결할 수 있습니다.

**체험 2: 환각 가능성**
아래를 입력해보세요:
```
EXEM의 2025년 4분기 매출을 알려줘.
```
→ Claude가 자신있게 답할 수 있지만, 정확하지 않을 수 있습니다.
→ 학습 데이터에 없는 정보를 "지어낼" 가능성이 있습니다.

**대응 연습:**
이제 이런 식으로 요청해보세요:
```
EXEM에 대해 알고 있는 것을 알려줘.
확실하지 않은 정보는 "불확실"이라고 표시해줘.
```

💡 "확실하지 않으면 모른다고 해줘"는 모든 AI 사용에서 유용한 패턴입니다.
   습관으로 만드세요.
