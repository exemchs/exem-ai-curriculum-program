# Day 2: 내 업무를 AI로 해보기

핵심 메시지: "AI는 도구가 아니라 역량(capacity)이다."
소요 시간: 약 60분

## 시각 규칙

- 섹션 헤더: `[N] 제목` + `═══` (굵은 선)
- 섹션 끝: `───` (가는 선)
- 닫힌 박스 금지 (한글 정렬 깨짐)
- 일러스트: "보여줘야 이해되는 것"에만 (영어 위주, 한글 설명은 바깥에)
- 아이콘: 비교, 흐름, 팁 등에 활용
- 프로그레스: 체크리스트 + `████░░░░` 바
- 인터랙션: 매 단계 AskUserQuestion

## 진행 방법 (Phase A/B 프로토콜)

각 Block은 2턴에 걸쳐 진행한다:

**Phase A (첫 번째 턴):**
1. 해당 Block의 reference 파일을 읽는다
2. EXPLAIN 섹션의 내용을 시각 요소와 함께 설명한다
3. EXECUTE 섹션의 실습을 안내한다
4. AskUserQuestion으로 진행 선택지를 제공한다:
   - "실습 완료! 다음으로"
   - "조금 더 시간이 필요해요"
   - "질문이 있어요"
5. **STOP — 반드시 여기서 멈추고 사용자 응답을 기다린다**

**Phase B (두 번째 턴):**
1. QUIZ 섹션을 읽고 AskUserQuestion으로 퀴즈를 낸다
2. 정답/오답 피드백을 준다
3. 프로그레스 바를 업데이트한다
4. 다음 Block Phase A로 넘어간다
