# Day 0: 오리엔테이션

처음 온 참가자를 위한 5분 안내.

## 시각 규칙

이 규칙은 Day 0 뿐 아니라 전체 커리큘럼에서 적용된다. 반드시 지켜야 한다.

- 섹션 헤더: `[N] 제목` 다음 줄에 `═══════════════════════════════` (굵은 선)
- 섹션 끝: `───────────────────────────────` (가는 선)
- 닫힌 박스 금지: 한글과 ASCII가 섞이면 오른쪽 테두리가 어긋난다. 절대 사용하지 않는다.
- 일러스트: "보여줘야 이해되는 것"에만 사용. 영어 위주로 그리고, 한글 설명은 바깥에 둔다.
- 아이콘: 비교, 흐름, 팁 등에 활용
- 프로그레스: 체크리스트 형태 + `████░░░░░░` 바
- 인터랙션: 매 단계마다 AskUserQuestion으로 진행 선택지 제공
- 설치 안내: 프로그램에서 반복하지 않음. 설치가 안 되어 있으면 아래 문구로 리디렉션:
  "랜딩 페이지에서 설치를 먼저 완료해주세요: https://exem-ai-curriculum-landing.vercel.app"

## 진행 방식

Day 0은 다른 Day와 다르게 **퀴즈 없이** 진행한다. 각 단계 끝에 AskUserQuestion으로 다음 진행을 묻는다.

### 진행 순서

1. references/orientation.md를 읽는다
2. EXEM ASCII 아트 배너를 출력한다:

```
███████╗██╗  ██╗███████╗███╗   ███╗
██╔════╝╚██╗██╔╝██╔════╝████╗ ████║
█████╗   ╚███╔╝ █████╗  ██╔████╔██║
██╔══╝   ██╔██╗ ██╔══╝  ██║╚██╔╝██║
███████╗██╔╝ ██╗███████╗██║ ╚═╝ ██║
╚══════╝╚═╝  ╚═╝╚══════╝╚═╝     ╚═╝
  A X   C A M P  ──  D a y  0
```

3. 인트로 텍스트를 출력한다:

```
이 프로그램은 EXEM 내부 구성원을 위해 설계된
Claude Code 학습 커리큘럼입니다.

CX그룹(비전공자) 눈높이에 맞춘 쉬운 난이도로 구성했으며,
코딩 경험이 없어도 괜찮습니다.

커리큘럼은 2026년 3월 기준 최신 AI 워크플로우 트렌드를
반영하여 설계되었습니다.

설계: 조현서, CX그룹장 @ EXEM
기준일: 2026.03.03
```

4. 인용구를 출력한다:

> "Claude Code 팀에서는 모두가 코딩합니다. PM도, 디자이너도, 매니저도."
> — Boris Cherny, Claude Code 창시자

5. [1]~[4] 단계를 references/orientation.md에서 읽고 순서대로 안내한다
6. 각 단계 끝에 AskUserQuestion으로 진행 여부를 묻는다
7. 프로그레스 바를 각 단계에서 업데이트한다:
   - [1] 완료: `██░░░░░░░░ 25%`
   - [2] 완료: `████░░░░░░ 50%`
   - [3] 완료: `██████░░░░ 75%`
   - [4] 완료: `████████░░ 100%`
8. 마지막에 Day 1 시작 안내로 마무리
