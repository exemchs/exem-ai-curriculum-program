# Day 0 오리엔테이션 내용

## 1. 이건 결국 대화입니다

ChatGPT 써보셨죠? 그것과 동일합니다.
질문하면 답변하고, 부탁하면 실행합니다.

다른 점 하나: Claude Code는 **내 컴퓨터 안에서** 동작합니다.
파일을 직접 읽고, 만들고, 수정합니다. 복사-붙여넣기가 필요 없습니다.

지금 이 대화 자체가 Claude Code입니다. 이미 시작한 겁니다.

## 2. Cursor 둘러보기

지금 보이는 화면이 Cursor입니다. 3군데만 기억하세요:

| 위치 | 이름 | 하는 일 |
|------|------|---------|
| 왼쪽 세로줄 | 파일 탐색기 | 프로젝트 파일 목록 (폴더를 열었다면) |
| 아래쪽 | 터미널 | Claude Code가 동작하는 곳 (지금 여기) |
| 가운데 | 에디터 | 파일 내용을 볼 수 있는 곳 |

> 파일 탐색기가 안 보이면 Cursor 메뉴 File → Open Folder → exem-ai-curriculum-program 선택

## 3. Claude와 첫 대화

지금 이 터미널에서 Claude와 대화하고 있습니다.
뭐든 물어봐도 됩니다:

- "EXEM이 뭐하는 회사야?"
- "이 프로젝트에 어떤 파일이 있어?"
- "오늘 날씨 알려줘" (이건 못합니다 — 인터넷 검색은 안 됩니다)

핵심: **구체적으로 말할수록 결과가 좋습니다.**
"도와줘" < "이 파일을 읽고 요약해줘" < "이 파일을 읽고 3줄로 요약해줘"

## 4. 멈추기 / 이어하기

**멈출 때:** "여기까지 정리해줘"라고 입력하세요. Claude가 진행 상황을 저장합니다. 그 후 터미널을 닫아도 됩니다.

**이어할 때:** 3단계입니다.
1. Cursor 열기 → Terminal → New Terminal
2. `cd exem-ai-curriculum-program && claude` 입력
3. "이어서 해줘" 입력

이게 전부입니다. 언제든 멈추고, 언제든 이어할 수 있습니다.
