# Day 0 오리엔테이션


## [1] 이건 결국 대화입니다
═══════════════════════════════

ChatGPT나 Claude 웹에서 대화해본 적 있죠?
Claude Code도 그것과 동일합니다. 질문하면 답하고, 부탁하면 실행합니다.

핵심 차이 하나:

  🌐 웹 Chat                     💻 Claude Code
  ─────────────                  ─────────────
  브라우저에서 대화               터미널에서 대화
  파일을 복사해서 붙여넣기        내 파일을 직접 읽고 수정
  대화가 끝나면 기억 사라짐       CLAUDE.md로 기억 유지
  매번 처음부터 설명              /명령어로 반복 가능

지금 이 대화 자체가 Claude Code입니다. 이미 시작한 겁니다.

───────────────────────────────


## [2] Cursor 안에서 뭘 보고 있는 건지
═══════════════════════════════

지금 보고 있는 화면이 Cursor입니다.
3곳만 기억하세요:

```
+------------------+------------------------+
|                  |                        |
|   EXPLORER       |       EDITOR           |
|                  |                        |
|   📁 .claude/    |   (file content here)  |
|   📁 docs/       |                        |
|   📄 CLAUDE.md   |                        |
|   📄 README.md   |                        |
|                  |                        |
+------------------+------------------------+
|                                           |
|              TERMINAL                     |
|   > You are talking to Claude here        |
|                                           |
+-------------------------------------------+
```

왼쪽: Explorer (파일 탐색기) — 프로젝트 파일 목록
가운데: Editor (에디터) — 파일 내용 표시
아래: Terminal (터미널) — Claude Code가 동작하는 곳 ← 지금 여기

💡 파일 탐색기가 안 보여도 괜찮습니다!
   지금은 터미널 대화에 집중하세요.
   나중에 파일을 눈으로 보고 싶으면
   Cursor 메뉴 → File → Open Folder → exem-ai-curriculum-program 선택
   (이때 터미널이 초기화되므로 Claude 세션이 끝난 후에 하세요)

## [3] Claude와 첫 대화
═══════════════════════════════

구체적으로 말할수록 결과가 좋습니다:

  😕 모호         "도와줘"
  🙂 보통         "이 파일을 읽고 요약해줘"
  🎯 구체적       "README.md를 읽고 핵심 내용을 3줄로 요약해줘"

💡 지금 바로 해보세요!
   아무거나 물어봐도 됩니다:
   - "이 프로젝트에 어떤 파일이 있어?"
   - "CLAUDE.md를 읽고 요약해줘"

## [4] 멈추기 / 이어하기
═══════════════════════════════

💬 멈출 때:

  1. "여기까지 정리해줘" 입력
  2. Claude가 진행 상황 저장 💾
  3. 터미널을 닫아도 OK 👋

🔄 이어할 때:

  1. Cursor 열기
  2. File → Open Folder → exem-ai-curriculum-program 선택
  3. Terminal → New Terminal
  4. `claude` 입력
  5. "이어서 해줘" 입력

언제든 멈추고, 언제든 이어할 수 있습니다.
