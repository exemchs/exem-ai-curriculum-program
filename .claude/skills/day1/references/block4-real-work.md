# Block 4: 내 업무 하나 해보기


## EXPLAIN

## [4] 내 업무 하나 해보기
═══════════════════════════════

Day 1 마무리입니다.
오늘 배운 것을 실제 업무에 적용해봅시다.

오늘 배운 3가지:

  🔹 Lv.1  CO-STAR로 구조화된 요청
  🔷 Lv.2  AI에게 역질문 시키기 (CRIT)
  🔶 Lv.3  CLAUDE.md로 시스템에 새기기

이 세 가지를 조합하면:
CLAUDE.md가 기본 맥락을 제공하고(Lv.3),
AI가 부족한 부분을 질문하고(Lv.2),
내가 구조화된 형식으로 답하면(Lv.1) → 최상의 결과

"AI를 도구로 쓰는 것에서, 동료로 일하는 것으로."

───────────────────────────────


## EXECUTE

### 실제 업무 적용

아래에서 하나를 골라 실제로 해보세요:

  📝 보고서 초안 작성
  📋 회의록 정리
  ✉️ 이메일/메시지 요약
  📊 데이터 정리
  🔍 리서치 + 요약

**진행 방법:**

1. 업무를 하나 선택합니다
2. CO-STAR 구조로 요청을 작성합니다:
   - C: 배경/상황
   - O: 원하는 결과
   - S: 형식
   - T: 톤
   - A: 대상
   - R: 출력 형식
3. 또는 CRIT 방식으로: "이 업무를 시작하기 전에 질문 5개 먼저 던져줘"
4. Claude Code로 실행합니다
5. 결과를 평가합니다: "이걸 실제로 쓸 수 있나?"

💡 완벽하지 않아도 괜찮습니다.
   중요한 건 "AI와 함께 일하는 감각"을 잡는 것입니다.
