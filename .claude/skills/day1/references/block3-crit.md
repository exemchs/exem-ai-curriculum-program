## [3] CRIT — AI에게 역질문 시키기
═══════════════════════════════

두 번째 레벨입니다.
"내가 다 지시하는 게 아니라, AI에게 역질문을 시켜라."

이런 식으로 시작합니다:
```
나는 [업무 주제]에 대한 보고서를 쓰려 해.
바로 쓰지 말고, 시작하기 전에 핵심 질문 5개를 먼저 던져줘.
```

왜 이게 더 좋을까요?

  CO-STAR (Lv.1)  내가 모든 걸 미리 생각해서 지시
                  → 내가 빠뜨린 건 AI도 빠뜨림

  CRIT (Lv.2)    AI가 먼저 빠진 부분을 질문
                  → 내가 생각 못한 것까지 커버

비유: 컨설턴트를 고용했을 때
- 나쁜 컨설턴트: 바로 보고서를 씀 → 방향이 어긋남
- 좋은 컨설턴트: 첫 미팅에서 질문부터 → 정확한 결과


### 예시: 이렇게 동작합니다

  나: "경쟁사 분석 보고서를 쓰려 해.
       바로 쓰지 말고 핵심 질문 5개를 먼저 던져줘."

  Claude:
  1. 어떤 경쟁사를 비교할까요? (전부? 특정 회사?)
  2. 비교 기준이 있나요? (가격, 기능, 시장점유율?)
  3. 보고서의 대상은 누구인가요? (경영진? 팀원?)
  4. 어떤 형식을 원하나요? (표? 슬라이드? 문서?)
  5. 분량은 어느 정도가 적당할까요?

  → 이 질문에 답한 후 AI가 작업하면
    처음부터 방향이 맞는 결과가 나옴
