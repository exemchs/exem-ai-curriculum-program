```
  Day 1                Day 2               Day 3
  컨텍스트              위임의 기술           도구 연결
  엔지니어링            ──────────          ──────────
  ──────────           AI에게 맡기고        MCP로 외부
  좋은 질문의           검증하는 법          도구를 연결
  구조를 배움
       │                    │                   │
       └────────────────────┴───────────────────┘

                         Day 4
                         자동화
                         ──────────
                         Skill + 검증 루프

                         Day 5
                         졸 업 🎓
                         ──────────
                         모든 것을 조합하여
                         나만의 AI 시스템 완성
```

### 내가 만든 것들

  📄 CLAUDE.md        → 내 비서의 업무 매뉴얼 (영구 기억)

  ⚡ Skill             → 반복 업무 자동화 (/명령어)
  🔍 검증 루프         → AI가 스스로 품질을 확인

이 네 가지가 합쳐지면:
**"프롬프트를 잘 쓰는 사람"이 아니라 "컨텍스트를 설계하는 사람"**
