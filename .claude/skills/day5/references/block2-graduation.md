# Block 2: 졸업 프로젝트 완성


## EXPLAIN

## [2] 졸업 프로젝트 완성
═══════════════════════════════

졸업 프로젝트: 본인 업무에 실제로 적용할 워크플로우 1개를 완성합니다.

### 졸업 프로젝트 구성

5일간 배운 모든 것을 조합합니다:

  📄 CLAUDE.md    → 맥락 (나는 누구, 어떤 규칙)
  🔗 MCP          → 도구 (외부 데이터 접근)
  ⚡ Skill         → 자동화 (/명령어로 실행)
  🔍 검증 루프     → 품질 (스스로 확인)


### 선택지

  A. Day 4에서 만든 Skill을 고도화
  ─────────────────────────────
  검증 루프 강화, MCP 연동 추가,
  더 구체적인 템플릿 등

  B. 새로운 워크플로우를 처음부터 만들기
  ─────────────────────────────
  다른 반복 업무를 Skill로 만들고
  CLAUDE.md + MCP + 검증 루프 적용

어떤 것을 선택해도 좋습니다.
핵심은 "실제로 쓸 수 있는 것"을 만드는 것입니다.

───────────────────────────────


## EXECUTE

### 졸업 프로젝트 제작 (30분)

**1단계: 주제 확정**
Day 4에서 만든 Skill을 고도화할지, 새로 만들지 정합니다.

**2단계: 설계**
Claude에게 계획을 세워달라고 요청합니다:
```
나는 [업무 설명]을 자동화하는 Skill을 만들려 해.
필요한 구성을 설계해줘:
- CLAUDE.md에 추가할 맥락
- 필요한 MCP 연결
- SKILL.md 구조
- 검증 루프 기준
```

**3단계: 제작**
설계대로 만듭니다. Claude에게 파일 생성을 요청하세요.

**4단계: 테스트**
만든 Skill을 실행해봅니다:
```
/[내스킬이름]
```

**5단계: 개선**
결과를 보고 부족한 점을 수정합니다.
검증 루프가 잘 동작하는지 확인합니다.

💡 완벽하지 않아도 됩니다.
   "실제로 쓸 수 있는 수준"이면 충분합니다.
   졸업 후에도 계속 개선할 수 있습니다.
