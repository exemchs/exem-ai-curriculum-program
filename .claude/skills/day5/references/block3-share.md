만든 것을 혼자 쓰기에는 아깝습니다.
팀과 공유하면 팀 전체의 AI 활용 수준이 올라갑니다.

### 공유하는 방법

  📁 Skill 공유
  ─────────────────────────────
  `.claude/skills/` 폴더가 Git에 포함되어 있으므로
  팀원이 같은 프로젝트를 clone하면
  모든 Skill을 바로 사용할 수 있습니다.

  내가 만든 `/weekly-report` Skill을
  팀원도 `/weekly-report`로 바로 실행 가능!

  📄 CLAUDE.md 공유
  ─────────────────────────────
  프로젝트 루트의 CLAUDE.md에 팀 규칙을 넣으면
  팀 전체의 AI 응답 품질이 일관되게 올라갑니다.

  예: "보고서는 항상 한국어, 표 형식, 3줄 요약 포함"
  → 팀원 누가 요청해도 같은 형식

## EXECUTE

### 팀 공유 준비

1. 만든 Skill을 팀에 공유할 준비를 합니다:
```
내가 만든 Skill의 사용법을 팀원에게 설명하는 문서를 만들어줘.
- Skill 이름과 용도
- 사용법 (/명령어)
- 예시 결과
```

2. (선택) CLAUDE.md에 팀 규칙 추가:
```
CLAUDE.md에 우리 팀의 공통 규칙을 추가해줘:
- [팀 규칙 1]
- [팀 규칙 2]
```
