# Block 2: 졸업 프로젝트 제작

## EXPLAIN

### 만들기 전에

Block 1에서 정한 3가지를 확인합니다:
1. **무엇을**: 어떤 업무를 자동화?
2. **어떻게**: 어떤 기술?
3. **결과물**: 어떤 형태?

### 만드는 순서

```
1. SKILL.md 뼈대 생성
2. 핵심 로직 작성 (수집 → 처리 → 출력)
3. 실행 테스트
4. 수정 → 재실행 (반복)
5. 완성
```

이 과정을 Claude와 대화하며 진행합니다.
코드를 직접 쓸 필요 없습니다. Claude에게 요청하면 됩니다.

### 팁

| 상황 | 해결 |
|------|------|
| 막혔을 때 | Claude에게 "지금 막힌 부분을 설명하면" 진단해줍니다 |
| 너무 복잡할 때 | 범위를 줄이세요. 핵심 기능 1개만 동작하면 성공 |
| MCP 연결이 안 될 때 | `/mcp`로 상태 확인 후 Claude에게 문의 |
| 결과가 엉뚱할 때 | SKILL.md의 지시를 더 구체적으로 수정 |

## EXECUTE

졸업 프로젝트를 만드세요:

```
졸업 프로젝트 스킬을 만들어줘.
.claude/skills/ 디렉토리에 만들어줘.

[Block 1에서 정한 내용을 여기에 설명]

단계별로 진행하면서 각 단계마다 확인시켜줘.
```

> 30분 안에 동작하는 뼈대를 완성하는 것이 목표입니다.
> 완벽하지 않아도 됩니다. "실행하면 뭔가 나온다" 수준이면 충분합니다.

완성 후 스킬을 실행하세요:
```
/[내 스킬 이름]
```
