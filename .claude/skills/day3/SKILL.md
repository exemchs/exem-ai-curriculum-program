# Day 3: 도구 연결하기 — MCP

핵심 메시지: "AI의 능력이 부족한 게 아니라, 맥락이 부족한 것이다."
소요 시간: 약 60분

## 출력 규칙 (중요)

- **토큰 절약**: Pro 플랜 사용자를 고려하여 출력을 최소화한다
- reference 파일을 그대로 복사하지 않는다. 핵심만 자연스럽게 전달한다
- 한 Block 설명은 **15줄 이내**로 한다
- 서술형 문장은 최소화하되, 시각 요소는 유지한다:
  헤더(`═══`), 종료선(`───`), 비교 테이블, 아이콘, 프로그레스 바
- 닫힌 박스 금지 (한글 정렬 깨짐)

## 진행 방법

각 Block은 다음 순서로 진행한다:

1. 해당 Block의 reference 파일을 읽는다
2. EXPLAIN 섹션의 내용을 시각 요소와 함께 설명한다 (예시 포함)
3. AskUserQuestion으로 선택지를 제공한다:
   - "직접 해볼래요" → EXECUTE 섹션의 실습을 안내한다
   - "이해했어요, 다음으로" → 바로 다음 Block으로 넘어간다
4. **STOP — 반드시 여기서 멈추고 사용자 응답을 기다린다**
5. 프로그레스 바를 업데이트하고 다음 Block으로 넘어간다
