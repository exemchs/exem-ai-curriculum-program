## [1] MCP 개념
═══════════════════════════════

Day 2에서 AI의 한계를 체험했죠?
인터넷 접근 불가, 외부 도구 사용 불가...
MCP는 이 한계를 해결합니다.

### MCP = AI에게 눈과 손을 달아주는 것

MCP(Model Context Protocol)는 AI와 외부 도구를 연결하는 규격입니다.

비유: USB-C

  🔌 USB-C가 하나의 포트로 모든 기기를 연결하듯
  🔗 MCP가 하나의 규격으로 모든 도구를 AI에 연결

```
                    MCP
                     |
        +------------+------------+
        |            |            |
     Notion      Calendar     GitHub
    (문서)       (일정)       (코드)
        |            |            |
        +------------+------------+
                     |
              Claude Code
           (하나의 대화에서 모두 접근)
```

Claude Code가 MCP를 통해 외부 도구의 데이터를 읽고 쓸 수 있게 됩니다.


### MCP 없는 AI vs MCP 있는 AI

  ❌ MCP 없을 때                  ✅ MCP 있을 때
  ─────────────                  ─────────────
  브라우저 탭 전환하며 작업       한 곳에서 모든 도구 접근
  데이터를 복사해서 붙여넣기      직접 데이터를 가져옴
  수동으로 확인하고 입력          자동으로 읽고 처리
  "캘린더 내용 알려줘" → 불가     "내일 일정 정리해줘" → 바로 실행

Phil Schmid(Hugging Face 엔지니어)의 말:
"마법은 더 똑똑한 모델이 아니라, 올바른 맥락 제공에 있다."

MCP는 AI를 더 똑똑하게 만드는 게 아니라,
AI에게 더 많은 맥락을 제공하는 것입니다.
