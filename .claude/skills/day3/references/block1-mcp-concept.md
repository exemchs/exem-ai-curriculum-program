### MCP = AI에 도구를 연결하는 표준 규격

MCP(Model Context Protocol)는 AI와 외부 도구를 연결하는 규격입니다.

비유: USB-C

  🔌 USB-C가 하나의 포트로 모든 기기를 연결하듯
  🔗 MCP가 하나의 규격으로 모든 도구를 AI에 연결

```
                    MCP
                     |
        +------------+------------+
        |            |            |
     Notion      Calendar     Figma
    (문서)       (일정)       (디자인)
        |            |            |
        +------------+------------+
                     |
              Claude Code
           (하나의 대화에서 모두 접근)
```

### 솔직한 이야기: MCP는 CLI 전용이 아닙니다

2026년 기준, ChatGPT도 Claude 웹도 MCP를 지원합니다.
MCP 자체는 더 이상 놀라운 기능이 아닙니다.

그렇다면 Claude Code에서 MCP를 쓰는 이유는?

  웹 AI의 MCP                    Claude Code의 MCP
  ─────────────                  ─────────────
  클라우드에서 연결               내 PC에서 직접 연결
  제공되는 도구만 사용             로컬 파일, 로컬 DB도 접근 가능
  매번 새로 설정                  프로젝트에 저장되어 팀이 공유

💡 핵심: MCP를 "어디서" 쓰느냐보다
   "어떤 도구를 연결하면 내 업무가 바뀌는지"가 중요합니다.

### EXEM에서 연결할 수 있는 도구

  ⭐⭐⭐ Notion — 문서 관리 (가장 안정적)
  ⭐⭐⭐ Google Calendar — 일정 관리 (전 직원)
  ⭐⭐   Figma — 디자인 협업 (Pro 이상, 하루 200회)
  ⭐⭐   GitHub — 코드 관리 (엔지니어)
  ⭐     Mattermost — 개발팀 확인 필요 (Community Edition이라 활용 범위 제한 가능)

## EXECUTE

### MCP 연결 체험

1. 현재 연결된 MCP 서버를 확인해봅시다:
```
/mcp
```

2. 본인에게 필요한 도구를 하나 연결해보세요.
   Claude에게 안내를 요청하면 됩니다:
```
Notion MCP 서버를 연결하고 싶어. 단계별로 안내해줘.
```
또는:
```
Google Calendar MCP 서버를 연결하는 법 알려줘.
```

💡 연결이 잘 안 되거나 지금 안 해도 괜찮습니다.
   MCP 연결은 나중에 필요할 때 해도 됩니다.
   다음 Block에서 더 중요한 이야기를 합니다.
