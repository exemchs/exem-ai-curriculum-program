# Block 2: 연결 체험 (자율 실습)


## EXPLAIN

## [2] 직접 연결해보기
═══════════════════════════════

MCP 연결은 간단합니다. 본인에게 필요한 도구를 하나 골라서 해보세요.

### 연결 명령어

```
claude mcp add [서버이름] -- [실행 명령어]
```

### 빠른 예시: Notion

```
claude mcp add notion -- npx -y @notionhq/notion-mcp-server
```
(Notion API 토큰이 필요합니다 — Settings → Integrations)

💡 연결 방법을 모르겠으면 Claude에게 물어보세요:
```
[도구 이름] MCP 서버를 연결하고 싶어. 단계별로 안내해줘.
```

───────────────────────────────


## EXECUTE

### 자율 실습

1. `/mcp` 입력해서 현재 상태 확인
2. 필요한 도구가 있으면 연결해보기
3. 연결 후 간단한 테스트:
   - Notion: "내 Notion에서 최근 문서 3개 알려줘"
   - Calendar: "내 캘린더에 오늘 일정 뭐 있어?"

💡 지금 연결 안 해도 괜찮습니다.
   나중에 필요할 때 하면 됩니다.
   다음 Block이 오늘의 핵심입니다.
