## EXPLAIN

## [3] 연결된 도구로 업무하기
═══════════════════════════════

MCP를 연결했으니, 이제 실제 업무에 활용해봅시다.

핵심: MCP를 연결하는 것 자체가 목적이 아닙니다.
연결된 도구로 **실제 업무를 더 효율적으로 처리**하는 것이 목적입니다.

### 활용 시나리오

  📝 Notion 연결 시
  ─────────────────────────────
  "이번 주 회의록을 요약해줘"
  "프로젝트 페이지에서 진행 상황을 정리해줘"
  "회의록을 읽고 액션 아이템만 뽑아줘"

  📅 Calendar 연결 시
  ─────────────────────────────
  "다음 주 일정을 정리해서 보여줘"
  "이번 주 미팅 중 준비가 필요한 것 알려줘"
  "수요일에 빈 시간이 있는지 확인해줘"

  🔗 조합 활용 (여러 도구 연결 시)
  ─────────────────────────────
  "캘린더에서 이번 주 미팅 목록을 가져와서,
   각 미팅 준비사항을 정리해줘"
  "Notion 회의록을 읽고 팔로업 이메일 초안 작성해줘"


💡 Day 2에서 배운 위임 프레임워크를 여기에도 적용하세요:
   1. 기준 정의 → 2. MCP 도구와 함께 위임 → 3. 검증

## EXECUTE

### 연결한 도구로 실제 업무 처리

Block 2에서 연결한 MCP 도구를 활용하여 실제 업무를 처리해봅시다.

**Notion을 연결했다면:**
```
내 Notion에서 최근 회의록을 찾아서
핵심 내용을 3줄로 요약하고,
다음 액션 아이템을 리스트로 정리해줘.
```

**Calendar를 연결했다면:**
```
이번 주 내 일정을 정리해서 보여줘.
각 일정별로 준비해야 할 것이 있으면 함께 알려줘.
```

**결과를 파일로 저장:**
```
방금 정리한 내용을 summary.md 파일로 저장해줘.
```

💡 Claude Code는 결과를 파일로 직접 저장할 수 있습니다.
   이것이 웹 Chat과의 또 다른 차이입니다.
   웹에서는 결과를 복사해서 따로 저장해야 하지만,
   Claude Code는 바로 파일로 만들어줍니다.
