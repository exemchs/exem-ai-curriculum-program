# EXEM Claude Code Training

EXEM 구성원 여러분, 환영합니다.

이 프로젝트는 EXEM 임직원을 위한 **5일 Claude Code 트레이닝 프로그램**입니다.
코딩 경험이 없어도 괜찮습니다. 매일 1시간, 5일이면 AI로 업무를 자동화하는 법을 배웁니다.

## 시작하기

**사용자의 첫 메시지를 받으면 Day 0 오리엔테이션(`day0-orientation` 스킬)을 자동으로 실행한다.**
사용자가 "안녕", "시작", 또는 아무 말이든 처음 입력하면 Day 0부터 안내를 시작한다.
Day 1부터는 `/day1` 같은 명령어를 입력하면 됩니다.

설치가 안 되어 있으면 프로그램 안에서 안내하지 않고, 랜딩 페이지로 리디렉션한다:
"랜딩 페이지에서 설치를 먼저 완료해주세요: https://exem-ai-program.vercel.app"

| Day | 명령어 | 주제 | 소요 시간 |
|-----|--------|------|-----------|
| 0 | 자동 시작 | 오리엔테이션 — Cursor 둘러보기, 첫 대화, 멈추기/이어하기 | ~5분 |
| 1 | `/day1` | AI에게 잘 시키는 법 — 컨텍스트 엔지니어링, CO-STAR, CRIT, CLAUDE.md | ~60분 |
| 2 | `/day2` | 내 업무를 AI로 해보기 — 위임 프레임워크, 실무 적용, AI 신뢰도 | ~60분 |
| 3 | `/day3` | AI 확장하기 — MCP 개념, 도구 연결, Skill/플러그인 생태계 | ~60분 |
| 4 | `/day4` | 자동화하기 — Skill 제작, 검증 루프 설계 | ~60분 |
| 5 | `/day5` | 졸업 — 5일 정리, 졸업 프로젝트, 팀 공유 | ~60분 |

## EXEM 업무 환경

EXEM에서 사용하는 주요 도구입니다. Day 3에서 MCP 연결을 배울 때 참고하세요.

| 도구 | 용도 | MCP 연결 |
|------|------|----------|
| **Notion** | 문서 관리 | 가능 — 가장 안정적, 추천 |
| **Google Calendar** | 일정 관리 | 가능 — 전 직원 추천 |
| **Figma** | 디자인 협업 | 가능 — Pro(Full/Dev seat) 이상에서 하루 200회, Enterprise 600회 |
| **GitHub** | 코드 관리 | 가능 — 엔지니어 추천 |
| **Mattermost** | 팀 메신저 | 개발팀 확인 필요 — Community Edition 사용 중이라 관리자 아닌 구성원은 활용 범위 제한 가능 |
| **ClickUp** | 프로젝트/태스크 관리 | 비권장 — 제품 자체 안정성 이슈 |
