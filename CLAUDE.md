# EXEM Claude Code Training

EXEM 구성원 여러분, 환영합니다.

이 프로젝트는 EXEM 임직원을 위한 **5일 Claude Code 트레이닝 프로그램**입니다.
코딩 경험이 없어도 괜찮습니다. 매일 1시간, 5일이면 AI로 업무를 자동화하는 법을 배웁니다.

## 시작하기

터미널에서 `/day1-start`를 입력하면 1일차가 시작됩니다.

| Day | 명령어 | 주제 | 소요 시간 |
|-----|--------|------|-----------|
| 1 | `/day1-start` | 설치 + 첫 대화 + CLAUDE.md | ~60분 |
| 2 | `/day2-connect` | MCP로 외부 도구 연결 | ~60분 |
| 3 | `/day3-skill` | 나만의 스킬 완성 | ~60분 |
| 4 | `/day4-apply` | 실무 적용 (Session Wrap, Content Digest) | ~60분 |
| 5 | `/day5-graduate` | 졸업 프로젝트 | ~65분 |

## EXEM 업무 환경

EXEM에서 사용하는 주요 도구입니다. Day 2에서 MCP 연결을 배울 때 참고하세요.

| 도구 | 용도 | MCP 연결 |
|------|------|----------|
| **Mattermost** | 팀 메시징 (커뮤니티 버전) | 가능 — 채널 메시지 수집, 요약에 활용 |
| **ClickUp** | 프로젝트/태스크 관리 | 가능하나 비권장 — 제품 자체 안정성 이슈로 체험 수준만 권장 |
| **Notion** | 문서 관리 | 가능 — 가장 안정적, 추천 |
| **GitHub** | 코드 관리 | 가능 — 엔지니어 추천 |
| **Google Calendar** | 일정 관리 | 가능 — 전 직원 추천 |

### ClickUp MCP 참고사항

ClickUp MCP 서버는 연결 자체는 가능하지만, ClickUp API의 안정성 이슈로 실무에서 MCP를 통한 자동화를 적극 활용하기는 어렵습니다.
Day 2에서는 "MCP가 외부 도구를 어떻게 연결하는지" 개념을 이해하는 데 초점을 맞추세요.
실습은 Notion이나 Google Calendar처럼 안정적인 서버로 진행하는 것을 권장합니다.
