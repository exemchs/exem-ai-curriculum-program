# EXEM Claude Code Training

EXEM 구성원을 위한 5일 Claude Code 트레이닝 프로그램입니다.

## 사전 준비

- [Cursor 에디터](https://cursor.com) 설치
- [Anthropic 계정](https://console.anthropic.com) 생성

## 설치

```bash
# 1. 이 프로젝트를 클론합니다
git clone https://github.com/pyhsudochs/exem-ai-curriculum-program.git

# 2. 프로젝트 폴더로 이동합니다
cd exem-ai-curriculum-program

# 3. Claude Code를 설치합니다 (아직 설치하지 않았다면)
# Mac / Linux
curl -fsSL https://claude.ai/install.sh | bash

# Windows (PowerShell)
irm https://claude.ai/install.ps1 | iex

# 4. Claude Code를 실행합니다
claude

# 5. 1일차를 시작합니다
/day1-start
```

## 커리큘럼

| Day | 주제 | 배우는 것 | 만드는 것 |
|-----|------|-----------|-----------|
| 1 | 시작하기 | Claude Code 설치, CLAUDE.md, 핵심 기능 | CLAUDE.md 규칙서 |
| 2 | 외부 도구 연결 | MCP 개념, 서버 연결 | Context Sync 스킬 뼈대 |
| 3 | 나만의 스킬 완성 | Subagent, 출력 포맷, 검증 | 완성된 스킬 1개 |
| 4 | 실무 적용 | Multi-agent, Quiz-First 학습 | Session Wrap + Content Digest |
| 5 | 졸업 | 종합 프로젝트 | 졸업 프로젝트 스킬 |
